    Client Applicatif

    Interrogation du serveur distant de Connexion

    On se connecte au websocket qui va se connecter au serveur d'authentification et retourner un token valide en cas de connexion réussit

    Example request:

    curl -X POST \
    "http://localhost/loginDistant" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -d '{"Email":"dignissimos","Password":"aliquid"}'

    const url = new URL(
    "http://localhost/loginDistant"
);

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "Email": "dignissimos",
    "Password": "aliquid"
}

fetch(url, {
    method: "POST",
    headers: headers,
    body: body
})
    .then(response => response.json())
    .then(json => console.log(json));

    Example response (200):

    {
    "token": "token",
    "token_expire": "14400"
}

    HTTP Request

    POST loginDistant

    Body Parameters

    Parameter Type Status Description
    Email String required L'email de l'utilisateur
    Password String required Mot de passe de l'utilisateur

    Algorithme décryptage : Cesar

    Fonction permettant de décrypter un message avec l'algorithme de Cesar. Dans un premier temps, on passe une par une les lettres à leurs suivantes (a devient b, etc...) Ensuite, si nous ne trouvons pas ladetaille dans la chaine totale de caractère, on recommence la procédure précédente avec cette nouvelle chaine

    Example request:

    curl -X POST \
    "http://localhost/loginDistant" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -d '{"Email":"dignissimos","Password":"aliquid"}'

    "http://localhost/loginDistant"

    method: "POST",
    headers: headers,
    body: body
})
    .then(response => response.json())
    .then(json => console.log(json));

    Parameters

    Parameter Type Description
    $chaine string Le message à décrypter
    $choix int Le début de la clé à tester
    $pos int Le sens du décalage (1 vers la droite, -1 vers la gauche)
    $mode int Cryptage ou Décryptage (ici Cryptage)
    $fin int La fin de la clé à tester
